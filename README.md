日曜19:00(JST,UTC+9)に月曜の接近をお知らせしてくれるDiscordのbot。間違えて消したので二代目。
https://discord.com/developers/ でトークンを取得し、token.jsonに適当に貼り付けて、cmdでPython bot.pyすればいい感じに動きます。
